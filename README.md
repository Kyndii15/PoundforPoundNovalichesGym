# PoundforPoundNovalichesGym
A web-based Gym Management System built as a capstone project for a gym in Novaliches, Quezon City. The system features role-based access for Admin, Manager, Coach, and Member, QR code attendance tracking, real-time dashboard metrics, email-based authentication, online membership payments, and complete administrative management workflows.
